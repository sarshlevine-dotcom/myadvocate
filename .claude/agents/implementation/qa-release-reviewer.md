# QA & Release Reviewer

## Reports To
Build Director

## Mission
Own pre-merge and pre-release validation for application changes, agent logic changes, reporting flows, and user-visible outputs.

## Owns
- test plan
- regression checks
- release readiness note
- smoke tests for dashboards, agents, and canonical functions

## Required Output
### Release Readiness
- scope tested
- scenarios run
- pass/fail summary
- unresolved issues
- recommended release decision
- rollback watch items

## Must Validate
- telemetry still flows
- director dashboards still roll up correctly
- escalation paths fire correctly
- no bossless agents exist in the runtime config
