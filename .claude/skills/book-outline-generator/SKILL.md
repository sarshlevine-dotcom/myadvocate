## Purpose

Generate structured outlines for nonfiction healthcare advocacy books that MyAdvocate can publish as a passive income and authority channel.

## Instructions

### Step 1 — Identify topic
Work with the founder to identify:
- Core subject (e.g., "The Patient's Guide to Fighting Insurance Denials")
- Target reader (patients who have received a denial, caregivers, healthcare advocates)
- Unique angle (what MyAdvocate's data and tools provide that no other book does)
- Target length (standard nonfiction: 50,000-80,000 words / 12-18 chapters)

### Step 2 — Generate chapter outline
Produce a complete outline with:
- Book title and subtitle
- Introduction chapter overview
- 10-15 chapter titles with 3-5 subheading points each
- Appendix suggestions (state rights reference, denial code glossary, letter templates)
- Target word count per chapter

### Step 3 — Define production path
Map the outline to existing MyAdvocate content:
- Which chapters can be adapted from blog articles?
- Which chapters require original research?
- Which chapters should reference specific MyAdvocate tools?

## Output Format

1. **Book Title:** [title: subtitle]
2. **Target Audience:** [description]
3. **Unique Value Proposition:** [what makes this book different]
4. **Chapter Outline:** [full numbered list with subpoints]
5. **Production Path:** [existing content to adapt vs. write fresh]
6. **Estimated Timeline:** [at X chapters/week, completion in Y months]

## Notes

- Phase 3+ priority — focus on product and content first
